What is coding?

Coding, or computer programming, is the composition of instructions that computers, and sometimes servers, can follow to perform tasks. It involves designing and implementing algorithms, step-by-step specifications of procedures, by writing code in one or more programming/coding languages. Proficient programming usually requires expertise in several different subjects, including knowledge of the application domain, details of programming languages, specialized algorithms, and formal logic.

How was coding made?

Coding/computer programming was first put into use in the 19th century by the British mathemetician and writer, Ada Lovelace, who created an algorithm for "Charles Babbage's Analytical Engine", which is known to be the first ever computer. This algorithm was used to calculate numbers, and is regarded as the "first ever computer program" by historians.

What are the main coding languages?

The main coding languages, or those that are most commonly used with ease, include HTML, CSS, and JS.

HTML, which stands for "HyperText Markup Language", is a standard "markup" language that is designed to be viewed on a webpage through a document, or ".html" file. It usually defines the content and structure of a webpage.

CSS, which stands for "Cascading Style Sheets", is a standard "style sheet" language that is usually used to specify the styling of an HTML document. CSS can be used inline, internally, or externally, with the "style" element.

An example of inline CSS is... 

  <h1 style="color: black"></h1>

An example of internal CSS is... 

  <style>
    color: black;
  </style>

And for external CSS, you must link a seperate file containing CSS, with the following code... 

  <head>
    <link rel="stylesheet" href="[insertfilename].css">
  </head>

Lastly, JS, which stands for "JavaScript", is a standard programming language that is usually used to modify the behavior of a web page. Like CSS, JS can be used inline, internally, or externally, with the "script" element.

What can coding be used for?

Coding/computer programming can be used for a wide variety of things, depending on the language of code. For example, the programming (or coding) language, "C#" (C Sharp), can be used for developing desktop applications, while the programming language, "Swift", can be used for developing applications for Apple devices. Outside of devices you'd expect coding to be used on/in, many vehicles with an engine control unit (or ECU) use coding too!

How can you code?

The first step to coding is having a code editor. Anything from the simple "Notes" app, to a professional code compiling software, will work. For this section of the presentation, I will be using CodePen (for compiling only).

To display content through code, you'll have to use HTML. When using HTML, you must include the "body" element. This will signify the body, or main text/code, produced with code. The content we'll be displaying is the word, "Hello".

  <body>  The start of the "body" element.
  
  <h1>Hello</h1>  This is the main code/text. When hosted onto a webpage as a ".html" file,
  the only text that will appear is the word, "Hello".
  
  </body>  The end of the "body" element (signified by the slash).

Now that we've got the text, we're going to style it with the "color" style, using inline CSS.

  <body>
  
  <h1 style="color: red;">Hello</h1>  The "style" element is being implemented within the title 
  headline (or "h1"). There are hundreds, if not thousands, of styles which can be placed within
  the "style" element.
  
  </body>
